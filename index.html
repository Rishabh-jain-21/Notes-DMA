Dynamic memory allocation

In primitice variables like int , float , char the memory is allocated at compile time in known as Static memory Allocation (It has fixed size once we create the memory block its size cant be changed)

Dynamic Memory Allocation can be defined as a procedure in which the size of a data structure (like Array) is changed during the runtime.

We use a library in c language for this called as (stdlib.h) . This library provide 4 major functino for DMA that are :
1) Malloc()
2) Calloc()
3) reslloc()
4) free()

Malloc

Used to allocate the memory to structure (User defined data_types)
It takes one unsigned argument (it is a psitive integer value)

How it allocates memory ?

On success , it return base address of memory block

On failure , it return null pointer
